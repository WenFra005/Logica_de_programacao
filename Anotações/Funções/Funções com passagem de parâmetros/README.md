# Funções com passagem de parâmetros
Funções podem receber valores como entrada (parâmetros) e operar sobre eles. Esses valores são passados quando a função é chamada.
Exemplo:
```python
def soma(a, b):
    return a + b
```
Chamando a função:
```python
resultado = soma(3,5)
print(resultado)
```
Resultado: ``8``
